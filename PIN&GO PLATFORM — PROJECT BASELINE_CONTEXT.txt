DASHBOARD MODULES

LocksPage
Purpose: Lock operations only

Functions:
- Sync Lock
- NFC Sync
- Swap Lock
- Open Lock Detail

LocksPage should NOT show:
- battery health
- gateway alerts
- operational risk
- system alerts

LockDetailPage
Purpose: Lock management

Functions:
- Swap Lock
- Refresh TTLock inventory
- NFC actions
- Lock metadata
